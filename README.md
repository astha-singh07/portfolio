# Astha Singh Portfolio

This is my **personal portfolio website** built using **HTML and CSS**.  
It showcases my **skills, projects, and contact information** in a beginner-friendly layout.

---

## Features
- Home section with name, course, and avatar  
- About section with hobbies and goals  
- Skills section (C, Python, HTML, CSS, AI/ML basics)  
- Projects section (Calculator, Portfolio, AI Chatbot)  
- Contact section with email and form  

---

## Technologies Used
- HTML5  
- CSS3  

---

## How to Run
1. Download or clone the repository  
2. Open `index.html` in a web browser
